---
title: Drawer
description: 从屏幕边缘滑入的侧边面板组件
---

抽屉（Drawer）从屏幕边缘滑入，用于在不离开当前页面的情况下展示辅助内容或操作面板。基于 Dialog 封装，继承标题、描述、页脚、关闭按钮等完整能力，并新增方向控制和内容滚动功能。

```tsx
import { Drawer, DrawerContent } from '@skyroc/web-ui';
```

## 何时使用

- 需要在不离开当前页面的情况下查看详情、编辑表单或执行操作。
- 侧边导航面板、设置面板、筛选面板等场景。
- 与 Dialog 的区分：Dialog 居中弹出适合确认/提示，Drawer 从侧边滑入适合承载更多内容或持续性操作。
- 与 BottomSheet 的区分：BottomSheet 从底部弹出更适合移动端交互，Drawer 从任意方向滑入适合桌面端。

## 方向

通过 `contentProps.side` 控制 Drawer 的滑入方向，默认为 `right`：

| 方向     | 说明         | 适用场景             |
| -------- | ------------ | -------------------- |
| `right`  | 从右侧滑入   | 详情面板、属性编辑   |
| `left`   | 从左侧滑入   | 导航菜单、侧边栏     |
| `top`    | 从顶部滑入   | 通知面板、搜索面板   |
| `bottom` | 从底部滑入   | 操作面板、快捷菜单   |

左右方向的 Drawer 默认宽度为 `w-3/4`，在 `sm` 断点以上限制为 `max-w-sm`；上下方向的 Drawer 占满水平宽度。

## 滚动

`scrollable` 控制内容区是否可滚动，默认为 `true`。当内容超出面板高度时自动出现滚动条。设为 `false` 则内容区不滚动，适合内容高度固定的场景。

## 标题与描述

Drawer 继承 Dialog 的 `title` 和 `description`，在面板顶部显示标题栏和描述文本。通过 `titleProps` 和 `descriptionProps` 可进一步定制子组件属性。

## 页脚

通过 `footer` 控制页脚内容：

- **默认**：显示 OK 按钮，文本可通过 `okText` 自定义，点击回调为 `onOk`。
- **自定义**：传入 ReactNode 完全替换页脚内容。
- **隐藏**：传入 `null` 或 `false` 隐藏页脚。
