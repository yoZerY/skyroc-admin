## 编写原则

- 示例优先展示业务结构，不堆满所有 props。
- 示例中的视觉密度应接近真实 C 端页面或官网页面，而不是纯展示 Demo。
- 示例只使用必要组件，避免为了展示而制造复杂度。
- 当场景需要完整 API 时，链接回对应组件文档。
